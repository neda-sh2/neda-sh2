# Hi, I'm Neda Shirzadi 👋

I'm a bioinformatics researcher with a background that spans both wet lab and computational biology. I work at the intersection of genomics, machine learning, and data science — building pipelines and tools to extract biological insight from sequencing data.

---

## 🧬 What I Work On

My work spans the full spectrum of modern bioinformatics — from raw sequencing reads to biological interpretation:

- **Single-cell & bulk RNA-seq** — differential expression (PyDESeq2), GSEA, scRNA-seq QC and clustering (Scanpy)
- **Genome analysis** — read alignment (BWA, HISAT2), variant calling (bcftools), CRISPR off-target detection
- **3D genomics** — Hi-C contact map analysis, A/B compartment identification, P(s) distance-decay modeling
- **Long-read sequencing** — PacBio IsoSeq isoform analysis, full-length transcript quantification
- **Machine learning** — classification (KNN, logistic regression, random forest, XGBoost), clustering (K-Means, DBSCAN), dimensionality reduction (PCA, UMAP, t-SNE)
- **Sequence modeling** — k-mer Markov chain models, GO term prediction from protein sequence features
- **NLP** — text feature engineering (BoW, N-grams, NER) on biomedical text
